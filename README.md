# Online-Examination-System
Online Examination System in PHP using CodeIgniter framework
(Must Have MySql and XampServer Installed in PC & Tested on Windows 11)
PHP Folder Name : OnlineExaminationCI

1. Create New Folder with above name in directory "C:\xampp\htdocs\"
2. Place all the above files in directory "C:\xampp\htdocs\OnlineExaminationCI"
3. Create Database as given in the "DATABASE FILE" folder in PHP My Admin. Database name Should be "onlinexaminationci"
4. Start Apache and MySql Server from XampServer and access it.
